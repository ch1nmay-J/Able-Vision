📱 AbleVision – AI-Powered Mobile Assistant for the Visually Impaired

AbleVision is an AI-powered Android application designed to assist visually impaired individuals by providing real-time scene understanding, navigation support, and voice-based interaction.

The goal of this project is to enhance independence, safety, and accessibility using modern mobile and AI technologies.

🚀 Features:
🎯 Real-Time Object Detection
Detects surrounding objects and provides audio-based descriptions
🗣️ Voice-Guided Navigation
Helps users navigate indoor and outdoor environments
📄 OCR (Text-to-Speech)
Reads text from images such as signboards and documents
🚧 Proximity Alerts
Alerts users about nearby obstacles using sound and vibration
🌐 Offline Mode
Works even without internet connectivity
🤖 Conversational AI
Enables natural voice-based interaction

🎯 Problem Statement:
Visually impaired individuals often face:

Difficulty navigating unfamiliar environments
Limited awareness of surroundings
Lack of unified assistive solutions

Existing tools solve isolated problems but lack integration and contextual intelligence.

💡 Solution:
AbleVision provides a unified platform combining:

Computer Vision
Speech Processing
AI-based Assistance

This allows users to:
Understand surroundings in real-time
Navigate safely and independently
Interact naturally with their environment

🧠 Technologies Used:
Android Development (Java/Kotlin)
Machine Learning / Computer Vision
OCR (Optical Character Recognition)
Speech-to-Text & Text-to-Speech APIs
Mobile Sensors (for proximity alerts)

📊 Use Cases:
Street crossing assistance
Indoor navigation (homes, offices)
Public transport navigation
Reading signs and labels
Crowd navigation

🔄 User Flow:
User opens the app
Activates voice assistant
Camera captures surroundings
AI processes and detects objects
App provides real-time voice feedback
